# 游戏机制

本章节详细介绍《开拓者：正义之怒》的核心游戏机制。

## 基础攻击加值 (BAB)

BAB (Base Attack Bonus) 是角色近战和远程攻击命中的基础值，共分为三档：

- **满 BAB** — 每级 +1（战士、野蛮人等）
- **中 BAB** — 每级 +0.75，向下取整（牧师、吟游诗人等）
- **半 BAB** — 每级 +0.5，向下取整（法师、术士等）

BAB 达到 +6、+11、+16 时获得额外攻击次数。

## 豁免检定 (Saving Throws)

豁免分为强韧 (Fortitude)、反射 (Reflex)、意志 (Will) 三种，每档分为两档：

- **强豁免 (Good)** — 2 + 等级 / 2，向下取整
- **弱豁免 (Poor)** — 等级 / 3，向下取整
