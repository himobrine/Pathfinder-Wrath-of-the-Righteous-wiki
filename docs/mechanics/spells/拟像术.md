| 属性 | 值 |
|------|-----|

| 仪式 | 否 |

| 施法距离 | 触碰 |

| 言语成分 | V |
| 姿势成分 | S |

| 持续时间 | 直至被解除 |

## 描述

你创造一个野兽或类人生物的拟像，该生物在整段施法时间内必须一直在你的10尺内。结束施法时，你同时触碰该生物以及和该生物同样体型的一堆冰或者雪，然后冰雪变化为该生物的拟像。拟像是一个生物。它使用拟像对象本体在施法时的游戏数据，但其类别为构装，最大生命值为本体最大生命值的一半，并且它无法施展拟像术。拟像对你和你所选定的生物态度友善。它服从你的命令并在战斗中你的回合里行动。拟像不能升级，也不能进行短休和长休。如果拟像受到伤害，它只能通过你在长休期间进行修理来回复生命值。拟像每恢复一点生命值就要消耗价值 100gp 的材料，在修理期间，它必须位于你5尺内。拟像会一直持续到其生命值降至 0，随后就变回冰雪并立刻融化。再次施放该法术时，任何你已用该法术创造出来的拟像都会立刻被摧毁。
