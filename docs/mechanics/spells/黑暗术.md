# 黑暗术

| 属性 | 值 |
|------|-----|
| 英文名 | Darkness |
| 环阶 | 2 环 |
| 学派 | 塑能 |
| 仪式 | 否 |
| 施法时间 | 动作 |
| 施法距离 | 60尺 |
| 专注 | 是 |
| 言语成分 | V |
| 姿势成分 | S |
| 材料成分 | M（蝙蝠毛与一块煤） |
| 昂贵材料 |  |
| 持续时间 | 至多10分钟 |

## 描述

你指定施法距离内一点，并在法术持续时间内使魔法性黑暗Darkness充斥以该点为中心半径15尺的球状区域。黑暗视觉Darkvision也无法看穿这这种黑暗，非魔法的光照也无法将其照亮。或者，你可以将此法术施展于一个未被着装或携带的物件上，使黑暗Darkness充斥以该物件为中心的15尺光环区域。将该物件用不透明的东西遮掩（比如用碗或头盔盖住）时，会将黑暗Darkness隔绝。如果法术区域与任何二环或更低环阶法术创造的明亮光照Bright Light或微光光照Dim Light区域重叠，则这些造光的法术将被解除。
