# 剑刃爆发

| 属性 | 值 |
|------|-----|
| 英文名 | Sword Burst |
| 环阶 | 戏法 |
| 学派 | 咒法 |
| 仪式 | 否 |
| 施法时间 | 1动作 |
| 施法距离 | 自身(5尺半径) |
| 专注 | 否 |
| 言语成分 | V |
| 姿势成分 | × |
| 材料成分 | × |
| 昂贵材料 |  |
| 持续时间 | 立即 |

## 描述

·你在瞬间制造出一圈半透明的利刃清扫你的周围。法术范围内除你以外的每一个生物都必须成功通过一次敏捷豁免，否则将受到1d6点力场伤害。·你到达5级时，该法术的伤害增加1d6（变为2d6）。11级时再加1d6（变为3d6），17级时再加1d6（变为4d6）。
