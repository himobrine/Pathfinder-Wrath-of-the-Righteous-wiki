# 星界投影

| 属性 | 值 |
|------|-----|
| 英文名 | Astral Projection |
| 环阶 | 9 环 |
| 学派 | 死灵 |
| 仪式 | 否 |
| 施法时间 | 1小时 |
| 施法距离 | 10尺 |
| 专注 | 否 |
| 言语成分 | V |
| 姿势成分 | S |
| 材料成分 | M（你必须为此法术影响的每个生物准备一枚价值 1000gp+ 的红锆石和一根价值 100gp+ 的银棒，作为该法术的耗材） |
| 昂贵材料 | 贵材 |
| 持续时间 | 直到被解除 |

## 描述

你和施法距离内至多八名自愿的生物将星界躯体投影到星光位面（如果你已经在星光位面，则法术立即结束）。目标原本的躯体会被留在原地陷入静止，它会陷入昏迷Unconscious状态，不需要空气和食物也不会衰老。目标的星界躯体几乎完美的保持了原本的形态，并保留你本体的游戏资料和随身物。星界躯体和你本体的主要区别是从你肩胛骨间向后伸出的一条银线。这条线可见长度仅有一尺。如果线被切断——只有明确注明可以切断这种银线的效应才可以将其切断，目标的身体和星界躯体都会死亡。你的星界躯体可以在星光位面任意旅行。当星界躯体离开星光位面时，你的本体和物品将沿着银线被传送，使你以本体的形态到达新位面。星界躯体承受的伤害或效应不会反映在你的本体上，反之亦然。如果某生物的本体或星界躯体的生命值降至 0，则该生物自身的投影效应终止。你可以以一个魔法动作解除法术，结束对所有目标的星界投影。法术终止时，如果目标没有死亡，受术生物会被拉回原本的躯体，并脱离静止状态。
