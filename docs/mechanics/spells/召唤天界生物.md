| 属性 | 值 |
|------|-----|
| 英文名 | Conjure Celestial |

| 施法距离 | 90尺 |
| 专注 | 是 |
| 言语成分 | V |
| 姿势成分 | S |
| 材料成分 | × |
| 昂贵材料 |  |
| 持续时间 | 至多1小时 |

## 描述

·你召唤一个挑战等级4或更低的天界生物，并使其显现在施法范围内一处你能看见且未被占据的空间。该天界生物在生命值降至0时，或者法术终止时消失不见。·被召唤的天界生物对你和你的伙伴保持友善关系。你为召唤出来的天界生物掷先攻权，并依此获得其自己的行动回合。该天界生物服从你发出的所有语言命令（不需要你作动作），前提是该命令必须不违反其所属阵营。如果你不发出任何命令，该生物会在敌对生物前保护自己，如果没有敌对生物则不执行任何动作。·该天界生物的具体数值由DM掌控。·升环施法效应。使用9环法术位施展该法术时，你将召唤一个挑战等级5或者更低的天界生物。
