# 再生术

| 属性 | 值 |
|------|-----|
| 英文名 | Regenerate |
| 环阶 | 7 环 |
| 学派 | 变化 |
| 仪式 | 否 |
| 施法时间 | 1分钟 |
| 施法距离 | 触碰 |
| 专注 | 否 |
| 言语成分 | V |
| 姿势成分 | S |
| 材料成分 | M（一个转经筒） |
| 昂贵材料 |  |
| 持续时间 | 1小时 |

## 描述

一名你触及的生物恢复4d8+15点生命值。在持续时间内，目标在其每回合开始时恢复1点生命值，且其失去的任何身体部位都将在2分钟后重新长出。
