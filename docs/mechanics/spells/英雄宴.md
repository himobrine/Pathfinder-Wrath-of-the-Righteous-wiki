| 属性 | 值 |
|------|-----|
| 英文名 | Heroes' Feast |
| 环阶 | 6 环 |
| 学派 | 咒法 |
| 仪式 | 否 |

| 施法距离 | 自身 |

| 言语成分 | V |
| 姿势成分 | S |
| 材料成分 | M（一个价值1000+gp 的镶饰宝石的碗，作为该法术的耗材） |

| 持续时间 | 立即 |

## 描述

你咒唤一场盛宴出现在你周围地面上未被占据的10尺立方空间。盛宴需要1小时来享用，并会在这段时间结束后消失不见，而且下述增益效应在这1小时结束前不会生效。你可以让至多十二个生物一同分享盛宴。所有享用了大餐的生物都会获得几种增益。这些生物对毒素伤害具有抗力，免疫中毒和恐慌状态，生命值上限也将提高 2d10 点，并获得等量的生命值。这些增益将持续 24 小时。
