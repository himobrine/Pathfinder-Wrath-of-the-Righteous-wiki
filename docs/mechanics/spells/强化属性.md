# 强化属性

| 属性 | 值 |
|------|-----|
| 英文名 | Enhance Ability |
| 环阶 | 2 环 |
| 学派 | 变化 |
| 仪式 | 否 |
| 施法时间 | 动作 |
| 施法距离 | 触碰 |
| 专注 | 是 |
| 言语成分 | V |
| 姿势成分 | S |
| 材料成分 | M（毛发或羽毛） |
| 昂贵材料 |  |
| 持续时间 | 至多1小时 |

## 描述

你触碰一名生物并在力量、敏捷、智力、感知、魅力中选择一种属性。法术持续时间内，目标在使用所选属性进行的属性检定上具有优势。升环施法效应Using a Higher-Level Spell Slot。使用的法术位每比二环高一环，就可以多指定一个目标。你可以为每个目标选择不同的属性。
