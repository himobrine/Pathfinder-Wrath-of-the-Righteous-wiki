| 属性 | 值 |
|------|-----|
| 英文名 | Pass without Trace |
| 环阶 | 2 环 |
| 学派 | 防护 |
| 仪式 | 否 |
| 施法时间 | 动作 |
| 施法距离 | 自身 |
| 专注 | 是 |
| 言语成分 | V |
| 姿势成分 | S |
| 材料成分 | M（槲寄生烧成的灰烬） |
| 昂贵材料 |  |
| 持续时间 | 至多1小时 |

## 描述

你在法术持续时间于30尺光环范围内散发出遮蔽灵光。位于灵光范围期间，你和每名你选择的生物在敏捷（隐匿）检定上获得+10加值，且不会留下任何踪迹。
