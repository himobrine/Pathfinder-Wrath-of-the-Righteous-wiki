# 轰雷剑

| 属性 | 值 |
|------|-----|
| 英文名 | Booming Blade |
| 环阶 | 戏法 |
| 学派 | 塑能 |
| 仪式 | 否 |
| 施法时间 | 1动作 |
| 施法距离 | 自身(5尺半径) |
| 专注 | 否 |
| 言语成分 | × |
| 姿势成分 | S |
| 材料成分 | M |
| 昂贵材料 | 一把至少价值1sp的近战武器 |
| 持续时间 | 1轮 |

## 描述

·你挥舞作为法术成分的那把武器，对你周围 5 尺范围内的一个生物发动一次近战武器攻击。如果攻击成功命中，目标会受到这次攻击的正常影响，并被包复在爆破性的能量之中，直到你的下回合开始。如果目标在这之前主动移动了 5 尺或更多，法术效应将立刻结束并使其受到 1d8 雷鸣伤害。·这道法术的伤害随着你的等级提升而增加。你到达 5 级时，这次近战攻击将造成额外 1d8 雷鸣伤害，并且目标进行移动时受到的伤害提升为 2d8。两个伤害骰都会在 11 级时增加 1d8（2d8 和 3d8），17 级时再加 1d8（3d8 和 4d8）。
