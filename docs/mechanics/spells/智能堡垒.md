# 智能堡垒

| 属性 | 值 |
|------|-----|
| 英文名 | Intellect Fortress |
| 环阶 | 3 环 |
| 学派 | 防护 |
| 仪式 | 否 |
| 施法时间 | 1动作 |
| 施法距离 | 30尺 |
| 专注 | 是 |
| 言语成分 | V |
| 姿势成分 | × |
| 材料成分 | × |
| 昂贵材料 |  |
| 持续时间 | 至多1小时 |

## 描述

·在法术持续时间内，你或一个施法距离内的可见生物获得对心灵伤害的抗性，同时在智力、感知和魅力豁免检定上具有优势。·升环施法效应。使用4环或更高法术位施展该法术时，你使用的法术位每比3环高一环，就可以多选择一个生物作为目标。而各目标之间距离不能超过30尺。
