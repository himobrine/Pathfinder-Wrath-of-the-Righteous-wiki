# 行动自如

| 属性 | 值 |
|------|-----|
| 英文名 | Freedom of Movement |
| 环阶 | 4 环 |
| 学派 | 防护 |
| 仪式 | 否 |
| 施法时间 | 动作 |
| 施法距离 | 触碰 |
| 专注 | 否 |
| 言语成分 | V |
| 姿势成分 | S |
| 材料成分 | M（一条皮带） |
| 昂贵材料 |  |
| 持续时间 | 1小时 |

## 描述

你触碰一个自愿生物，并使目标在法术持续时间内移动时不受困难地形Difficult Terrain限制，并且任何法术或其他魔法效应都不能降低目标的速度Speed，或者使目标陷入麻痹Paralyzed或束缚Restrained状态。目标还会获得等同于其速度的游泳速度。该目标还可以花费5尺移动力直接从非魔法的束缚中挣脱（诸如镣铐或生物使其陷入的受擒Grappled状态）。升环施法效应Using a Higher Level Spell Slot。使用的法术位每比四环高一环，就可以额外再指定一名生物作为法术目标。
