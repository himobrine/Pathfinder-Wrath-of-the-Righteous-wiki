# 回生术

| 属性 | 值 |
|------|-----|
| 英文名 | Revivify |
| 环阶 | 3 环 |
| 学派 | 死灵 |

| 施法时间 | 动作 |
| 施法距离 | 触碰 |
| 专注 | 否 |
| 言语成分 | V |
| 姿势成分 | S |
| 材料成分 | M（一颗价值300gp+的钻石，作为该法术的耗材） |
| 昂贵材料 | 贵材 |
| 持续时间 | 立即 |

## 描述

你接触在前一分钟内刚刚死亡的一名生物。该生物以1点生命值回生。该法术不能复活老死的生物，也不能恢复失去的身体部位。
