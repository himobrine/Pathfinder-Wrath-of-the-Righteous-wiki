# 跳跃术

| 属性 | 值 |
|------|-----|
| 英文名 | Jump |
| 环阶 | 1 环 |
| 学派 | 变化 |
| 仪式 | 否 |
| 施法时间 | 附赠动作 |
| 施法距离 | 触碰 |
| 专注 | 否 |
| 言语成分 | V |
| 姿势成分 | S |
| 材料成分 | M（一只蚱蜢的后腿） |
| 昂贵材料 |  |
| 持续时间 | 1分钟 |

## 描述

你触碰一个自愿生物。在法术结束前，每个他的回合内一次该生物可以消耗10尺移动力来跳跃30尺。升环施法效应：使用二环或更高法术位施展该法术时，你使用的法术位每比一环高一环，就可以额外指定一个生物作为目标。
