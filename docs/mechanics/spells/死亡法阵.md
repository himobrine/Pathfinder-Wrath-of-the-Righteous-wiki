# 死亡法阵

| 属性 | 值 |
|------|-----|
| 英文名 | Circle of Death |
| 环阶 | 6 环 |
| 学派 | 死灵 |
| 仪式 | 否 |
| 施法时间 | 动作 |
| 施法距离 | 150尺 |
| 专注 | 否 |
| 言语成分 | V |
| 姿势成分 | S |
| 材料成分 | M（价值500gp+的研磨过的黑珍珠粉末） |
| 昂贵材料 | 贵材 |
| 持续时间 | 立即 |

## 描述

一阵负能量波从施法距离内某点放出，冲击 60 尺半径的球状区域。区域内的生物必须进行一次体质豁免。豁免失败者将受到 8d8点暗蚀伤害，豁免成功受到半数伤害。升环施法效应：使用七环或更高法术位施展该法术时，你使用的法术位每比六环高一环，法术的伤害就增加 2d8。
