# 箭如雨下

| 属性 | 值 |
|------|-----|
| 英文名 | Conjure Barrage |
| 环阶 | 3 环 |
| 学派 | 咒法 |
| 仪式 | 否 |
| 施法时间 | 动作 |
| 施法距离 | 自身 |
| 专注 | 否 |
| 言语成分 | V |
| 姿势成分 | S |
| 材料成分 | M（一把价值1CP+的近战或远程武器） |
| 昂贵材料 | 贵材 |
| 持续时间 | 立即 |

## 描述

你挥舞作为施法材料的武器，令相似的幽灵武器（或是适合这把武器的弹药）显现，它们将向前射出并在之后消失。每名由你选择且你可见的位于60尺锥形区域内的生物进行一次敏捷豁免检定，豁免失败者将受到5d8的力场伤害，豁免成功则只受一半伤害。升环施法效应。使用的法术位每比三环高一环，法术造成的伤害就增加1d8。
