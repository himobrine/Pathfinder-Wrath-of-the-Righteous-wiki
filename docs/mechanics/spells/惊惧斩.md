# 惊惧斩

| 属性 | 值 |
|------|-----|
| 英文名 | Staggering Smite |
| 环阶 | 4 环 |
| 学派 | 惑控 |
| 仪式 | 否 |
| 施法时间 | 附赠动作，当你使用近战武器或徒手打击命中一个生物后立即执行 |
| 施法距离 | 自身 |
| 专注 | 否 |
| 言语成分 | V |
| 姿势成分 |  |
| 材料成分 | × |
| 昂贵材料 |  |
| 持续时间 | 立即 |

## 描述

被该次攻击检定命中的目标会在此次攻击中额外受到 4d6 的心灵伤害。该目标必须进行一次感知豁免，豁免失败则目标直到你的下回合结束陷入震慑Stunned状态。升环施法效应：使用高于4环的法术位施放本法术时，你使用的法术位每比4环高一环，额外伤害就增加1d6
