# 命运宠儿

| 属性 | 值 |
|------|-----|
| 英文名 | Fortune’s Favor |
| 环阶 | 2 环 |
| 学派 | 预言 |
| 仪式 | 否 |
| 施法时间 | 1分钟 |
| 施法距离 | 60尺 |
| 专注 | 否 |
| 言语成分 | V |
| 姿势成分 | S |
| 材料成分 | M |
| 昂贵材料 | 一颗至少价值100GP的白色珍珠，在施法时被消耗 |
| 持续时间 | 1小时 |

## 描述

·你给予你自己或施法距离内你可见的一个自愿生物以潜藏的幸运。在法术结束之前，当被选中的生物进行一次攻击检定，属性检定或者豁免检定时，他能够使该法术结束并扔一颗额外的D20骰，并决定使用哪一颗D20。或者，当这个生物成为一个攻击检定的目标时，他能够使该法术结束并扔一颗额外的D20骰，并决定使用哪一颗D20:攻击者所掷骰的或通过该法术所掷骰。·若原D20骰有优势或劣势，那么通过该法术掷出的D20在原D20的优势或劣势生效后生效。·升环施法效应。使用3环或更高法术位施展该法术时，每比2环高一环的法术位可以令你指定一个额外的生物。
