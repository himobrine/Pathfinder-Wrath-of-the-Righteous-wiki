# 水下呼吸

| 属性 | 值 |
|------|-----|
| 英文名 | Water Breathing |
| 环阶 | 3 环 |
| 学派 | 变化 |
| 仪式 | 是 |
| 施法时间 | 动作 |
| 施法距离 | 30尺 |
| 专注 | 否 |
| 言语成分 | V |
| 姿势成分 | S |
| 材料成分 | M（一根短芦苇） |
| 昂贵材料 |  |
| 持续时间 | 24小时 |

## 描述

在法术终止前，该法术让施法距离内你能看见的至多十个自愿生物获得在水下呼吸的能力。此外，受术者依然保有正常的呼吸能力。
