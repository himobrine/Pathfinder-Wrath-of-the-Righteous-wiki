# 物件定位术

| 属性 | 值 |
|------|-----|
| 英文名 | Locate Object |
| 环阶 | 2 环 |
| 学派 | 预言 |
| 仪式 | 否 |
| 施法时间 | 动作 |
| 施法距离 | 自身 |
| 专注 | 是 |
| 言语成分 | V |
| 姿势成分 | S |
| 材料成分 | M（一根分叉的树枝） |
| 昂贵材料 |  |
| 持续时间 | 至多10分钟 |

## 描述

你点名或描述一个你熟悉的物件。只要该物件与你相距不超过 1000 尺，你就会感测到该物件的方位。如果该物件正在移动，则你也将知晓其移动的方向。该法术可定位一个你认识的特定物件，不过你需要在 30 尺内近距离看到过该物件至少一次；或者它也可定位特定的一类物件（如某种服装、珠宝、家具、工具或武器）中距你最近个体。如果有任意厚度的铅，即使只是薄薄一片铅片阻挡了你与被定位物件间的通路，则该物件也无法被该法术所定位。
