# 创造篝火

| 属性 | 值 |
|------|-----|
| 英文名 | Create Bonfire |
| 环阶 | 戏法 |
| 学派 | 咒法 |
| 仪式 | 否 |
| 施法时间 | 1动作 |
| 施法距离 | 60尺 |
| 专注 | 是 |
| 言语成分 | V |
| 姿势成分 | S |
| 材料成分 | × |
| 昂贵材料 |  |
| 持续时间 | 至多1分钟 |

## 描述

·你在距离之内你能看到的地面上燃起一堆篝火。直到此法术终结，该魔法篝火会占满5立方尺的空间。当你施放此法术时，所有在篝火所在空间的生物必须通过一次敏捷豁免，否则受到1d8点伤害。当生物在一个回合中第一次进入篝火所在的空间，或在篝火所在的空间结束其回合时，其同样必须进行该豁免。·篝火可以点燃区域内未被穿戴或者携带的可燃物体。·你到达5级时，该法术的伤害增加1d8（变为2d8）。11级时再加1d8（变为3d8），17级时再加1d8（变为4d8）。
