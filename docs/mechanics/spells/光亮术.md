| 属性 | 值 |
|------|-----|

| 施法距离 | 触碰 |
| 专注 | 否 |
| 言语成分 | V |
| 姿势成分 |  |
| 材料成分 | M（一只萤火虫或一片磷光苔藓） |
| 昂贵材料 |  |
| 持续时间 | 1小时 |

## 描述

你触碰一个大型或者更小，且未被他人携带或着装的物件。在法术终止前，物件将在其 20 尺半径范围内发出明亮光照，并在其外 20 尺范围发出微光光照。光的颜色由你决定。该物件被不透明的东西完全遮盖时，其光照被遮挡。如果你再次施展该法术，则该法术终止。
