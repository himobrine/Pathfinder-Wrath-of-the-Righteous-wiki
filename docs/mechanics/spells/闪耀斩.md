# 闪耀斩

| 属性 | 值 |
|------|-----|
| 英文名 | Shining Smite（Branding Smite） |
| 环阶 | 2 环 |
| 学派 | 变化 |
| 仪式 | 否 |
| 施法时间 | 附赠动作，当你使用近战武器或徒手打击命中一个生物后立即执行 |
| 施法距离 | 自身 |
| 专注 | 是 |
| 言语成分 |  |
| 姿势成分 |  |
| 材料成分 | 言 |
| 昂贵材料 |  |
| 持续时间 | 至多1分钟 |

## 描述

被该次攻击检定命中的目标会在此次攻击中额外受到2d6点光耀伤害。目标会发出5尺半径的明亮光照直至法术结束。在此期间，对其进行的攻击检定具有优势，且其不能从隐形Invisible状态中受益。升环施法效应：使用高于二环的法术位施放本法术时，你使用的法术位每比2环高一环，额外伤害就增加1d6。
