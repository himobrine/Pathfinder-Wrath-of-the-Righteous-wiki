# 颤栗之触

| 属性 | 值 |
|------|-----|
| 英文名 | Chill Touch |
| 环阶 | 戏法 |
| 学派 | 死灵 |
| 仪式 | 否 |
| 施法时间 | 动作 |
| 施法距离 | 触碰 |
| 专注 | 否 |
| 言语成分 | V |
| 姿势成分 | S |
| 材料成分 | × |
| 昂贵材料 |  |
| 持续时间 | 立即 |

## 描述

你引导出来自坟墓的刺骨阴冷，对目标进行一次近战法术攻击。命中时，目标受到1d10暗蚀伤害，并且直至你下一回合的结束前都无法恢复生命值。戏法升级：你到达 5 级时，该法术的伤害增加 1d10（变为 2d10）。11 级时再加 1d10（变为 3d10），17 级时再加 1d10（变为 4d10）。
