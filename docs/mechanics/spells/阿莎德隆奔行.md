| 属性 | 值 |
|------|-----|

| 言语成分 | V |
| 姿势成分 | S |
| 材料成分 | × |
| 昂贵材料 |  |
| 持续时间 | 至多1分钟 |

## 描述

·巨龙汹涌的火焰从你的脚边喷涌而出，赋予你爆炸般的速度。法术持续时间内，你的速度提升20尺，并且移动不会触发借机攻击。·当你移动至一个生物或未被着装或携带的物件5尺内时，其被你的灼热尾迹灼烧，受到1d6点火焰伤害。一个生物或物件每回合只会承受一次此伤害。·升环施法效应。当你用4环或更高的法术位施展该法术时，你使用的法术位每比3环高一环，你的移动速度额外增加5尺，法术造成的伤害便增加1d6。
