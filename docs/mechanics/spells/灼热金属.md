# 灼热金属

| 属性 | 值 |
|------|-----|
| 英文名 | Heat Metal |
| 环阶 | 2 环 |
| 学派 | 变化 |
| 仪式 | 否 |
| 施法时间 | 动作 |
| 施法距离 | 60尺 |
| 专注 | 是 |
| 言语成分 | V |
| 姿势成分 | S |
| 材料成分 | M（一片铁片和一团火焰） |
| 昂贵材料 |  |
| 持续时间 | 至多1分钟 |

## 描述

指定施法距离内一件你能看见的人造金属物件（例如一把金属武器，或者一套金属质的重甲或中甲），并使该物件变得炽热。施展该法术时，任何与该物件有物理接触的生物均受到 2d8 的火焰伤害。直到法术终止前，只要这个物件依然在施法距离内，你都可以在你随后的每个回合在以一个附赠动作来再次造成此伤害。若握持或着装该物件的生物因此受到伤害，则该生物必须通过一次体质豁免，否则如果这个生物只要能丢下该物件，该生物就必须丢下该物件。如果该物件没有被丢下，则直至你的下一回合开始为止，该生物进行攻击检定和属性检定时具有劣势。升环施法效应：使用高于二环的法术位施展该法术时，你使用的法术位每比二环高一环，伤害就增加1d8。
