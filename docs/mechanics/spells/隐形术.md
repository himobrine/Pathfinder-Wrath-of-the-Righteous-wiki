# 隐形术

| 属性 | 值 |
|------|-----|
| 英文名 | Invisibility |
| 环阶 | 2 环 |
| 学派 | 幻术 |
| 仪式 | 否 |
| 施法时间 | 动作 |
| 施法距离 | 触碰 |
| 专注 | 是 |
| 言语成分 | V |
| 姿势成分 | S |
| 材料成分 | M（一根包裹在阿拉伯胶中的睫毛） |
| 昂贵材料 |  |
| 持续时间 | 至多1小时 |

## 描述

你触碰的一个生物进入隐形状态并维持至法术终止。如果目标进行攻击检定、造成伤害或施展法术，则该法术提前终止。升环施法效应：使用三环或更高法术位施展该法术时，你使用的法术位每比二环高一环，就可以额外指定一个生物作为目标。
