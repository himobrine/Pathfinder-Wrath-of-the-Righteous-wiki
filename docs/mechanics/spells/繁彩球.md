# 繁彩球

| 属性 | 值 |
|------|-----|
| 英文名 | Chromatic Orb |
| 环阶 | 1 环 |
| 学派 | 塑能 |
| 仪式 | 否 |
| 施法时间 | 动作 |
| 施法距离 | 90尺 |
| 专注 | 否 |
| 言语成分 | V |
| 姿势成分 | S |
| 材料成分 | M（一枚价值 50gp+ 的钻石） |
| 昂贵材料 | 贵材 |
| 持续时间 | 立即 |

## 描述

你向施法距离内某个目标掷出一颗能量球。为所创造的球体从强酸、寒冷、火焰、闪电、毒素、雷鸣中选择一种类型然后对该目标发动一次远程法术攻击。攻击命中时，该目标将受到 3d8 点你所选类型的伤害。如果你的两枚及以上d8骰子投出相同的数字，能量球跳向原目标30尺内你所选择的另一个目标。对新目标进行一次攻击检定，然后进行新的伤害掷骰。除非你以高于二环的法术位施展法术，否则能量球不会继续跳跃。升环施法效应：使用高于一环的法术位施展该法术时，你使用的法术位每比一环高一环，法术的伤害就增加 1d8。能量球能够最多跳跃等于消耗的法术位数目的次数，并且每次施展该法术时，一个生物只能被指定为目标一次。
