# 卓姆吉瞬间召唤

| 属性 | 值 |
|------|-----|
| 英文名 | Drawmij’s Instant Summons |
| 环阶 | 6 环 |
| 学派 | 咒法 |
| 仪式 | 是 |
| 施法时间 | 1分钟 |
| 施法距离 | 触碰 |
| 专注 | 否 |
| 言语成分 | V |
| 姿势成分 | S |
| 材料成分 | M（一颗价值1000gp+的蓝宝石） |
| 昂贵材料 | 贵材 |
| 持续时间 | 直至被解除 |

## 描述

你触摸施展法术时使用的蓝宝石与一件最重不超过10磅，最长不超过6尺的物件。法术在物件上留下一个隐形Invisible的标记，也在蓝宝石上留下隐形的物件名称。你每次施展该法术时都必须使用不同的蓝宝石作为其材料成分。在那之后，你可以用一个魔法Magic动作唤出该物件的名称并粉碎这颗蓝宝石，该物件将无视任何距离和位面的限制立即出现在你的手上，法术随即终止。如果另一个生物正持握或携带着该物件，则粉碎蓝宝石无法将该物品送过来。不过你会知晓是谁占有了该物品，以及其此刻身处何处。
