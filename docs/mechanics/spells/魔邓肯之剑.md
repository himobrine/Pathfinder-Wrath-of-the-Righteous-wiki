# 魔邓肯之剑

| 属性 | 值 |
|------|-----|
| 英文名 | Mordenkainen's Sword |
| 环阶 | 7 环 |
| 学派 | 塑能 |
| 仪式 | 否 |
| 施法时间 | 动作 |
| 施法距离 | 90尺 |
| 专注 | 是 |
| 言语成分 | V |
| 姿势成分 | S |
| 材料成分 | M（一个微缩模型的剑，价值至少250gp） |
| 昂贵材料 | 贵材 |
| 持续时间 | 至多1分钟 |

## 描述

你在施法距离内创造出一个悬浮的幽灵剑，其在法术持续时间内持续存在。幽灵剑出现时，你可以对该剑 5 尺内的一个目标进行一次近战法术攻击。命中时，目标受到 4d12+你的施法属性调整值 点的力场伤害。在你的后续回合内，你可以以一个附赠动作将剑移动至多 30 尺至一个你能看见的位置。随后你可以对相同或不同目标再次发动该攻击。
