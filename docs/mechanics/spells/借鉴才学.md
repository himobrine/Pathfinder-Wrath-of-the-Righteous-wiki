# 借鉴才学

| 属性 | 值 |
|------|-----|
| 英文名 | Borrowed Knowledge |
| 环阶 | 2 环 |
| 学派 | 预言 |
| 仪式 | 否 |
| 施法时间 | 1动作 |
| 施法距离 | 自身 |
| 专注 | 否 |
| 言语成分 | V |
| 姿势成分 | S |
| 材料成分 | M |
| 昂贵材料 | 一本价值至少25gp的书 |
| 持续时间 | 1小时 |

## 描述

·你从往昔的魂灵中获取知识。选择一项你没有熟练的技能。在法术的持续时间里，你拥有所选技能的熟练项。如果你再次释放该法术，该法术会提前结束。
