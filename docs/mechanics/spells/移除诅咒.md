# 移除诅咒

| 属性 | 值 |
|------|-----|
| 英文名 | Remove Curse |
| 环阶 | 3 环 |
| 学派 | 防护 |
| 仪式 | 否 |
| 施法时间 | 动作 |
| 施法距离 | 触碰 |
| 专注 | 否 |
| 言语成分 | V |
| 姿势成分 | S |
| 材料成分 | × |
| 昂贵材料 |  |
| 持续时间 | 立即 |

## 描述

你触及一名生物或一个物件，使影响其的所有诅咒终止。如果该物件是一件诅咒魔法物品，则其上的诅咒不会因此解除，不过该法术可以打破诅咒物品与其所有者之间的同调Attunement，使其所有者可以将其卸下或丢弃。
