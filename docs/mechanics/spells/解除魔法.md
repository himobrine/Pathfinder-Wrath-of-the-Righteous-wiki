| 属性 | 值 |
|------|-----|
| 英文名 | Dispel Magic |

| 施法距离 | 120尺 |
| 专注 | 否 |
| 言语成分 | V |
| 姿势成分 | S |
| 材料成分 | × |
| 昂贵材料 |  |
| 持续时间 | 立即 |

## 描述

在施法距离内指定一名生物、一个物件或一处魔法效应。所有影响该目标的三环或更低环阶法术即告终止。每个影响目标的四环或更高环阶法术都需要以你的施法属性进行一次属性检定（DC 10+目标法术环阶），检定成功时目标法术终止。升环施法效应Using a Higher-Level Spell Slot。你直接终止影响目标的小于等于该施法环阶的法术效应。
