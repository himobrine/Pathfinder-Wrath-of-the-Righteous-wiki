# 塔莎腐蚀酿

| 属性 | 值 |
|------|-----|
| 英文名 | Tasha’s Caustic Brew |
| 环阶 | 1 环 |
| 学派 | 塑能 |
| 仪式 | 否 |
| 施法时间 | 1动作 |
| 施法距离 | 自身(30尺线状) |
| 专注 | 是 |
| 言语成分 | V |
| 姿势成分 | S |
| 材料成分 | M |
| 昂贵材料 | 一点腐烂的食物 |
| 持续时间 | 至多1分钟 |

## 描述

·一股30尺长，5尺宽的的线状强酸从你的位置向你指定的任意方向喷射而出。在线状区域内的每个生物必须进行一次敏捷豁免，豁免失败者将在法术持续时间内被强酸覆盖，或直到自己或其他生物用一个动作从其身上刮下或者洗下。被强酸覆盖的生物在每个自己回合开始时受到2d4点强酸伤害。·升环法术效应。使用2环或更高法术位施展该法术时，你使用的法术位每比1环高一环，法术的伤害就增加2d4。
