| 属性 | 值 |
|------|-----|

| 姿势成分 | S |
| 材料成分 | × |
| 昂贵材料 |  |

## 描述

一把你触碰的非魔法武器变成魔法武器。选择下述伤害类型之一：强酸、寒冷、火焰、闪电、雷鸣。法术持续时间内，那把武器在攻击检定上具有+1加值，且在命中时造成额外1d4点你所选伤害类型的伤害。升环施法效应Using a Higher-Level Spell Slot。使用五环或六环法术位时，攻击检定的加值变为+2，额外伤害变为2d4。使用七环或更高法术位时，攻击检定的加值变为+3，额外伤害变为3d4。
