# 回避侦测

| 属性 | 值 |
|------|-----|
| 英文名 | Nondetection |
| 环阶 | 3 环 |
| 学派 | 防护 |
| 仪式 | 否 |
| 施法时间 | 动作 |
| 施法距离 | 触碰 |
| 专注 | 否 |
| 言语成分 | V |
| 姿势成分 | S |
| 材料成分 | M（一撮价值25gp+的钻石尘，作为该法术的耗材） |
| 昂贵材料 | 贵材 |
| 持续时间 | 8小时 |

## 描述

法术持续时间内，你使你触摸的目标不会被预言Divination法术发现。目标可以是一名自愿生物，或者是在任何方向上都不长于10尺的地点或物件。目标不能被任何预言Divination法术选为目标，也不能被魔法探知传感器感测到。
