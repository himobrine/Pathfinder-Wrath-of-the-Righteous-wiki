|------|-----|

| 施法距离 | 90尺 |
| 专注 | 是 |
| 言语成分 | V |
| 姿势成分 | S |
| 材料成分 | M（三颗坚果壳） |

| 持续时间 | 至多1分钟 |

## 描述

你选择施法距离内一点为源点，10尺球状区域内的每个生物必须成功通过一次感知豁免，否则目标不能执行反应和附赠动作，并且在每个回合开始时掷一次d10根据下表内容来决定本回合的行为。1d10	当回合行为1	该目标本回合不执行任何动作，它使用其所有移动力向一个随机方向移动。你可以掷一次 d4 决定其方向：1北2东3南4西。2-6	该生物本回合不移动也不执行任何动作。7-8	该生物不进行移动，使用其动作对可触及范围内一个随机生物发动一次近战攻击。如果没有生物在其触及范围内，则该生物本回合什么都不做。9-10	该生物自行决定当回合行为。升环施法效应：使用五环或更高法术位施展该法术时，你使用的法术位每比四环高一环，球状区域半径就增加 5 尺。
