# 引力裂沟

| 属性 | 值 |
|------|-----|
| 英文名 | Gravity Sinkhole |
| 环阶 | 4 环 |
| 学派 | 塑能 |
| 仪式 | 否 |
| 施法时间 | 1动作 |
| 施法距离 | 120尺 |
| 专注 | 否 |
| 言语成分 | V |
| 姿势成分 | S |
| 材料成分 | M |
| 昂贵材料 | 一块黑色的大理石 |
| 持续时间 | 立即 |

## 描述

·一道半径20尺球形范围的破坏性能量在施法距离内你能看见的一点为源点形成，并开始拖拽范围内的生物。所有在范围内的生物必须进行一次体质豁免检定。若豁免失败，该生物受到5d10点力场伤害并且会被直直拉向能量的中心，并在离中心尽可能近的未被占据的空间停止（哪怕这会让他飞到天上）。若豁免成功，则其受到一半伤害并且不会被拉走。·升环施法效应。当你使用5环或者更高环位的法术位施展该法术时，每比4环高出一环的法术位可以使该法术的伤害增加1d10。
