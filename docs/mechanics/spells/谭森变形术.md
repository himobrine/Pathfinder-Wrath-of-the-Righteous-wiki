| 属性 | 值 |
|------|-----|
| 英文名 | Tenser's Transformation |
| 环阶 | 6 环 |
| 学派 | 变化 |
| 仪式 | 否 |
| 施法时间 | 1动作 |
| 施法距离 | 自身 |
| 专注 | 是 |
| 言语成分 | V |
| 姿势成分 | S |
| 材料成分 | M |
| 昂贵材料 | 一缕公牛毛发 |
| 持续时间 | 至多10分钟 |

## 描述

·你以魔法为燃料赋予自己耐久和武艺。直到法术结束之前，你不能施展法术，并且得到如下增益：·你得到50点临时生命。如果这些临时生命保持到了法术结束，它们会消失。·你在使用简易或军用武器时，攻击检定具有优势。·当你用武器攻击击中一个目标，它将受到额外2d12力场伤害。·你熟练于所有盔甲，盾牌，简易武器和军用武器。·你熟练于力量豁免和体质豁免。·当你在自己的回合采取攻击动作时，你可以攻击两次而非一次。如果已经有类似的能力给予你额外攻击，例如额外攻击Extra Attack，你无视这项增益。·在法术结束时你需要立即成功通过一次DC15的体质豁免检定，否则将承受1级力竭。
