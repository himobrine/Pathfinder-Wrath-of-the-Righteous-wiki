# 秘法锁

| 属性 | 值 |
|------|-----|
| 英文名 | Arcane Lock |
| 环阶 | 2 环 |
| 学派 | 防护 |
| 仪式 | 否 |
| 施法时间 | 动作 |
| 施法距离 | 触碰 |
| 专注 | 否 |
| 言语成分 | V |
| 姿势成分 | S |
| 材料成分 | M（价值 25gp+ 的金粉，作为该法术的耗材） |
| 昂贵材料 | 贵材 |
| 持续时间 | 直到被解除 |

## 描述

你触碰关上的一扇门，一扇窗，一个容器或出入口，并使目标在法术持续时间内被魔法闭锁。这种闭锁无法被任何非魔法方式解除。你和你施展法术时允许的其他生物能够无视秘法锁打开和关闭被封锁的物件。你还可以设置一条密语，在该锁闭物件 5 尺内念出密语时，秘法锁将被压制 1 分钟。
