| 属性 | 值 |
|------|-----|

| 昂贵材料 |  |

## 描述

你在施法距离内指定一点，以该点为中心将一处边长 5 尺的立方区域召唤出旋转的匕首。区域内的每个生物受到4d4挥砍伤害。生物同样会在其进入立方区域、在立方区域中结束回合或立方区域移动进入其占据空间时受到此伤害。一个生物一回合中只会受到一次该伤害。在你后续的回合中，你可以使用一个魔法动作将立方区域传送最多30尺距离。升环施法效应：使用高于二环的法术位施展该法术时，你使用的法术位每比二环高一环，法术的伤害就增加 2d4。
