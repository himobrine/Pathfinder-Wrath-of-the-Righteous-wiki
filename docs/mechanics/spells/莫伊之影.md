| 属性 | 值 |
|------|-----|
| 英文名 | Shadow of Moil |
| 环阶 | 4 环 |
| 学派 | 死灵 |

| 施法时间 | 1动作 |
| 施法距离 | 自身 |
| 专注 | 是 |
| 言语成分 | V |
| 姿势成分 | S |
| 材料成分 | M |
| 昂贵材料 | 至少价值150gp的镶嵌在宝石里的不死生物的眼睛 |
| 持续时间 | 至多1分钟 |

## 描述

·如同火焰般的暗影完全笼罩住你的身体，使你对于其他人处于重度遮蔽，直到法术结束。·暗影将你周围10尺内的微光照明转变为黑暗，或者把同样区域的明亮照明转变为微光。直到法术结束之前，你得到对光耀伤害的抗性。此外，不论何时在你周围10尺内的生物用攻击命中你时，暗影会猛击那个生物，对他造成2d8点黯蚀伤害。
