|------|-----|

| 姿势成分 | S |
| 材料成分 | × |

| 持续时间 | 至多1小时 |

## 描述

·你的魔法加深了生物对自己才华的理解。你触碰一个自愿生物，并给予它你选择的一项技能的专精；直到法术结束之前，生物每次在使用选择的技能进行属性检定时都可以将它的熟练值加倍。·你所选择的技能必须是目标已经熟练的并且没有受益于其他类似将熟练值加倍的效果，例如专精Expertise。
