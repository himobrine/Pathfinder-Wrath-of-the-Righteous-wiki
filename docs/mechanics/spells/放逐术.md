# 放逐术

| 属性 | 值 |
|------|-----|
| 英文名 | Banishment |
| 环阶 | 4 环 |
| 学派 | 防护 |
| 仪式 | 否 |
| 施法时间 | 动作 |
| 施法距离 | 30尺 |
| 专注 | 是 |
| 言语成分 | V |
| 姿势成分 | S |
| 材料成分 | M（一颗五芒星） |
| 昂贵材料 |  |
| 持续时间 | 至多1分钟 |

## 描述

一个在你施法距离内你可见的生物必须进行一次魅力豁免，豁免失败者会在法术持续期间被转移到一个无害的半位面，身处半位面期间，目标会陷入失能Incapacitated状态。当法术终止时，目标将重新出现在其被放逐前的位置。如果该位置被占据，则目标将出现在距离该位置最近的一个未被占据空间。若该目标是一只异怪，一个天界生物，一个元素生物，一个精类或一只邪魔，则在法术持续一分钟后，目标不会返回。该目标会被改为转移到与其生物类型相关的一个位面（由DM决定）中的随机地点。升环施法效应。使用高于四环的法术位施展此法术时，你使用的法术位每比四环高一环，就能多指定一个生物。
