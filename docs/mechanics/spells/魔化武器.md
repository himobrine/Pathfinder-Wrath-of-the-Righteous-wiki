| 属性 | 值 |
|------|-----|

| 言语成分 | V |
| 姿势成分 | S |
| 材料成分 | × |
| 昂贵材料 |  |
| 持续时间 | 1小时 |

## 描述

你触碰一件非魔法武器。直到法术结束前，它成为魔法武器且在进行攻击检定和伤害掷骰时具有+1 加值。你再次施放它时，该法术会提前结束。升环施法效应Using a Higher Level Spell Slot。使用三至五环的法术位时，加值增加至+2，使用高于五环的法术位时，加值增加至+3。
