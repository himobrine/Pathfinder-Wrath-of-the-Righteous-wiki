# 激愤斩

| 属性 | 值 |
|------|-----|
| 英文名 | Wrathful Smite |
| 环阶 | 1 环 |
| 学派 | 死灵 |
| 仪式 | 否 |
| 施法时间 | 附赠动作，当你使用近战武器或徒手打击命中一个生物后立即执行 |
| 施法距离 | 自身 |
| 专注 | 否 |
| 言语成分 | V |
| 姿势成分 |  |
| 材料成分 | × |
| 昂贵材料 |  |
| 持续时间 | 1分钟 |

## 描述

被该次攻击检定命中的目标会在此次攻击中额外受到1d6点暗蚀伤害，并且目标必须通过一次感知豁免检定，否则直到法术结束为止陷入恐慌Frightened状态。在自己的每回合结束时，被恐慌Frightened的目标可以重复此豁免，成功则结束该法术对自身的影响。升环施法效应Using a Higher-Level Spell Slot。使用的法术位每比一环高一环，伤害就提高1d6。
