# 预言术

| 属性 | 值 |
|------|-----|
| 英文名 | Divination |
| 环阶 | 4 环 |
| 学派 | 预言 |
| 仪式 | 是 |
| 施法时间 | 动作 |
| 施法距离 | 自身 |
| 专注 | 否 |
| 言语成分 | V |
| 姿势成分 | S |
| 材料成分 | M（价值25gp+的熏香，作为该法术的耗材） |
| 昂贵材料 | 贵材 |
| 持续时间 | 立即 |

## 描述

此法术让你得以与一名神祇或其仆人取得联络。你可以就某个未来7日内将要发生的特定目标、事件或活动提出一个问题。DM将给予你真实的答复，这个答复可能是一段短语或一节神秘的短诗。此法术的回答并不会顾及所有可能影响结果的情况。比如其他法术的影响。若你在完成一次长休Long Rest之前施展多于一次该法术，则法术有概率不给出任何结果。此概率从第二次施展起每次累计25％。
