# 祝福术

| 属性 | 值 |
|------|-----|
| 英文名 | Bless |
| 环阶 | 1 环 |
| 学派 | 惑控 |
| 仪式 | 否 |
| 施法时间 | 动作 |
| 施法距离 | 30尺 |
| 专注 | 是 |
| 言语成分 | V |
| 姿势成分 | S |
| 材料成分 | M（一枚价值5GP+的圣徽） |
| 昂贵材料 | 贵材 |
| 持续时间 | 至多1分钟 |

## 描述

你祝福施法范围内至多3个生物。在法术终止前，每当受术目标进行攻击检定或豁免检定时，都可以在检定结果中额外加入1d4。升环施法效应。使用高于一环的法术位施展此法术时，你使用的法术位每比一环高一环，就能多指定一个生物。
