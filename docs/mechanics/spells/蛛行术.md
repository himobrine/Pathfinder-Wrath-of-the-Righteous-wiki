# 蛛行术

| 属性 | 值 |
|------|-----|
| 英文名 | Spider Climb |
| 环阶 | 2 环 |
| 学派 | 变化 |
| 仪式 | 否 |
| 施法时间 | 动作 |
| 施法距离 | 触碰 |
| 专注 | 是 |
| 言语成分 | V |
| 姿势成分 | S |
| 材料成分 | M（一滴沥青和一只蜘蛛） |
| 昂贵材料 |  |
| 持续时间 | 至多1小时 |

## 描述

你触碰一个自愿的生物，并使其在法术终止前能够在垂直表面上下或横向移动，其甚至可以在天花板上倒挂着移动而不需要用到双手。另外，目标获得与步行速度相等的攀爬速度。升环施法效应：使用高于二环法术位施放本法术时，你使用的法术位每比2环高一环，便能额外指定一个生物为目标。
