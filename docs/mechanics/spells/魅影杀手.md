# 魅影杀手

| 属性 | 值 |
|------|-----|
| 英文名 | Phantasmal Killer |
| 环阶 | 4 环 |
| 学派 | 幻术 |
| 仪式 | 否 |
| 施法时间 | 动作 |
| 施法距离 | 120尺 |
| 专注 | 是 |
| 言语成分 | V |
| 姿势成分 | S |
| 材料成分 | × |
| 昂贵材料 |  |
| 持续时间 | 至多1分钟 |

## 描述

你指定施法距离内一名你可见的生物，窥探其噩梦，依此创造一道只有目标能看见的幻象。目标生物进行一次感知豁免。豁免失败时，目标受到4d10点心灵伤害，且在法术持续时间内在属性检定和攻击掷骰上具有劣势。豁免成功，则目标只受一半伤害，法术终止。法术持续时间内，目标在其每回合结束时进行一次感知豁免，豁免失败则再次受到心灵伤害。豁免成功则法术终止。升环施法效应Using a Higher-Level Spell Slot。使用的法术位每比四环高一环，伤害就增加1d10。
