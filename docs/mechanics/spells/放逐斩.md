# 放逐斩

| 属性 | 值 |
|------|-----|
| 英文名 | Banishing Smite |
| 环阶 | 5 环 |
| 学派 | 咒法 |
| 仪式 | 否 |
| 施法时间 | 附赠动作，当你使用近战武器或徒手打击命中一个生物后立即执行 |
| 施法距离 | 自身 |
| 专注 | 是 |
| 言语成分 | V |
| 姿势成分 |  |
| 材料成分 | × |
| 昂贵材料 |  |
| 持续时间 | 至多1分钟 |

## 描述

被该次攻击检定命中的目标会在此次攻击中额外受到5d10力场伤害。如果此次攻击将目标的生命值削减到50点或更低，则目标必须进行一次魅力豁免，豁免失败则会在法术持续期间被转移到一个无害的半位面。身处半位面期间，目标会陷入失能Incapacitated状态。当法术终止时，目标将重新出现在其被放逐前的位置。如果该位置被占据，则目标将出现在距离该位置最近的一个未被占据空间。
