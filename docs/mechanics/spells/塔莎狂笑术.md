# 塔莎狂笑术

| 属性 | 值 |
|------|-----|
| 英文名 | Tasha’s Hedious Laughter |
| 环阶 | 1 环 |
| 学派 | 惑控 |
| 仪式 | 否 |
| 施法时间 | 动作 |
| 施法距离 | 30尺 |
| 专注 | 是 |
| 言语成分 | V |
| 姿势成分 | S |
| 材料成分 | M（一块甜馅饼和一根羽毛） |
| 昂贵材料 |  |
| 持续时间 | 至多1分钟 |

## 描述

施法距离内你可见的一名生物进行一次感知豁免。豁免失败，目标在持续时间内陷入失能Incapacitated和倒地Prone状态。在此期间，若目标能够发笑，则它会不受控制地狂笑，且无法结束自身的倒地Prone状态。目标在其每回合结束或受到伤害时，可以再进行一次感知豁免。如果豁免是因受到伤害所致，则该次豁免具有优势。豁免成功，法术终止。升环施法效应Using a Higher-Level Spell Slot。使用的法术位每比一环高一环，就可以额外指定一个目标。
