# 万箭齐发

| 属性 | 值 |
|------|-----|
| 英文名 | Conjure Volley |
| 环阶 | 5 环 |
| 学派 | 咒法 |
| 仪式 | 否 |
| 施法时间 | 动作 |
| 施法距离 | 150尺 |
| 专注 | 否 |
| 言语成分 | V |
| 姿势成分 | S |
| 材料成分 | M（一把价值1CP+的近战或远程武器） |
| 昂贵材料 | 贵材 |
| 持续时间 | 立即 |

## 描述

你挥舞作为施法材料的武器并选择施法距离内一点。数以百计相似的幽灵武器（或是适合这把武器的弹药）齐射而出，并在之后消失。以那一点为中心，半径 40 尺、高 20 尺的柱状区域内的每名由你选择且你可见的生物进行一次敏捷豁免检定，豁免失败者将受到8d8的力场伤害，豁免成功则只受一半伤害。
