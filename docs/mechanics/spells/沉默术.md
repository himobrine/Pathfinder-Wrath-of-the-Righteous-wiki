| 属性 | 值 |
|------|-----|

| 专注 | 是 |
| 言语成分 | V |
| 姿势成分 | S |
| 材料成分 | × |
| 昂贵材料 |  |
| 持续时间 | 至多10分钟 |

## 描述

你指定施法距离内一点，以该点为中心半径 20 尺的球状区域在法术持续时间内将无法产生任何声音，且外界的声音也无法穿行于该区域内。任何完全位于该区域内的生物和物件都将免疫雷鸣伤害，而任何生物在完全位于该区域内时都将陷入耳聋Deafened状态。此外，该区域内无法施展包含言语Verbal成分的法术。
