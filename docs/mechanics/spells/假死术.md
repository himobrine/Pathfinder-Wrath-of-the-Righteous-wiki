| 属性 | 值 |
|------|-----|
| 英文名 | Feign Death |
| 环阶 | 3 环 |
| 学派 | 死灵 |
| 仪式 | 是 |
| 施法时间 | 动作 |
| 施法距离 | 触碰 |
| 专注 | 否 |
| 言语成分 | V |
| 姿势成分 | S |
| 材料成分 | M（一撮墓土） |
| 昂贵材料 |  |
| 持续时间 | 1小时 |

## 描述

你触碰一名自愿生物并将其变得如死去一样的僵直状态。在法术持续时间内，对目标进行的任何肉体检查和法术检测都显示目标已处于死亡状态。该状态下的目标陷入目盲Blinded以及失能Incapacitated状态，且其速度降至0。目标对除了心灵伤害外的所有伤害类型具有抗性，也免疫中毒状态。
