| 属性 | 值 |
|------|-----|

| 言语成分 | V |
| 姿势成分 |  |
| 材料成分 | × |
| 昂贵材料 |  |
| 持续时间 | 立即 |

## 描述

在施展法术期间一直位于施法距离内的至多五名生物获得等同于一次短休Short Rest的增益，并且恢复2d8点生命值Hit Point。一名生物直到完成一次长休Long Rest为止不能再次被此法术影响。升环施法效应Using a Higher-Level Spell Slot。使用的法术位每比二环高一环，治疗量就增加1d8。
