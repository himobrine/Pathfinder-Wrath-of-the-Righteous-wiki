# 迅捷箭袋

| 属性 | 值 |
|------|-----|
| 英文名 | Swift Quiver |
| 环阶 | 5 环 |
| 学派 | 变化 |
| 仪式 | 否 |
| 施法时间 | 附赠动作 |
| 施法距离 | 自身 |
| 专注 | 是 |
| 言语成分 | V |
| 姿势成分 | S |
| 材料成分 | M（一个箭袋，价值至少1gp） |
| 昂贵材料 | 贵材 |
| 持续时间 | 至多1分钟 |

## 描述

你施展该法术时和在法术结束前以一个附赠动作，可以用一个射出箭矢或弩矢的武器进行两次攻击，例如长弓或轻弩。法术会魔法性地为每次攻击生成所需的弹药。以此法生成的每发箭矢或弩矢如同类非魔法弹药一般造成伤害，且会在命中或失手后立即瓦解。
