# 目盲/耳聋术

| 属性 | 值 |
|------|-----|
| 英文名 | Blindness/Deafness |
| 环阶 | 2 环 |
| 学派 | 变化 |
| 仪式 | 否 |
| 施法时间 | 动作 |
| 施法距离 | 120尺 |
| 专注 | 否 |
| 言语成分 | V |
| 姿势成分 |  |
| 材料成分 | × |
| 昂贵材料 |  |
| 持续时间 | 1分钟 |

## 描述

处于你施法范围内的一个你可见的生物必须进行一次体质豁免，豁免失败者在法术持续期间陷入目盲Blinded或耳聋Deafened状态（由你选择）。目标可以在其每个回合结束时重复此豁免，豁免成功则终止在其身上的此法术。升环施法效应。使用高于二环的法术位施展此法术时，你使用的法术位每比二环高一环，就能多指定一个生物。
