## 描述

该法术在施法距离内一棵大型Large或更大的未活化植物与同位面上任意距离外的另一棵植物之间创造一道魔法连接。施法时，你必须见过或触碰过出口植物至少一次。在持续时间内，任何生物都可以花费5尺移动力进入目标植物内，然后从出口植物处出来。木遁术Transport via Plants和树跃术Tree Stride将树木转变为一个运输网络。
