| 属性 | 值 |
|------|-----|
| 英文名 | Searing Smite |
| 环阶 | 1 环 |

| 仪式 | 否 |

| 施法距离 | 自身 |
| 专注 | 否 |
| 言语成分 | V |
| 姿势成分 |  |
| 材料成分 | × |
| 昂贵材料 |  |
| 持续时间 | 1分钟 |

## 描述

被该次攻击检定命中的目标会在此次攻击中额外受到1d6火焰伤害，并被魔法性火焰引燃。直至法术终止前，目标会在每个其回合开始时受到1d6点火焰伤害，并进行一次体质豁免。豁免失败则法术继续生效，成功则法术终止。升环施法效应：使用高于一环的法术位施放本法术时，你使用的法术位每比1环高一环，此法术造成的所有伤害都增加1d6。
