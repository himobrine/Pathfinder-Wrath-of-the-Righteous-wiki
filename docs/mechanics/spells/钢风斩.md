# 钢风斩

| 属性 | 值 |
|------|-----|
| 英文名 | Steel Wind Strike |
| 环阶 | 5 环 |
| 学派 | 咒法 |
| 仪式 | 否 |
| 施法时间 | 动作 |
| 施法距离 | 30尺 |
| 专注 | 否 |
| 言语成分 |  |
| 姿势成分 | S |
| 材料成分 | M（一把价值1×p+ 的近战武器） |
| 昂贵材料 | 贵材 |
| 持续时间 | 立即 |

## 描述

你舞动用于施法的武器，随后忽然消失并发起袭击，恰如一阵钢铁的疾风。选择施法距离内最多 5 个你可见的生物。向每个生物分别进行一次近战法术攻击。若击中，目标受到 6d10 力场伤害。在这之后，你可以传送到法术目标之一的周围 5 尺内，你可以看见的未被占据的空间。
