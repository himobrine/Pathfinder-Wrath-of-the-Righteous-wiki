# 灾祸术

| 属性 | 值 |
|------|-----|
| 英文名 | Bane |
| 环阶 | 1 环 |
| 学派 | 惑控 |
| 仪式 | 否 |
| 施法时间 | 动作 |
| 施法距离 | 30尺 |
| 专注 | 是 |
| 言语成分 | V |
| 姿势成分 | S |
| 材料成分 | M（一滴血） |
| 昂贵材料 |  |
| 持续时间 | 至多1分钟 |

## 描述

你选择施法距离内至多三个你可见的生物，迫使其分别进行一次魅力豁免。若有目标在此次豁免中失败，则在法术结束前，该目标进行的攻击检定或豁免检定都必须从检定结果中减去一个d4。升环施法效应。使用高于一环的法术位施展此法术时，你使用的法术位每比一环高一环，就能多指定一个生物。
