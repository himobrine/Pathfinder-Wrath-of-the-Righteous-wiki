# 不灭明焰

| 属性 | 值 |
|------|-----|

| 施法时间 | 动作 |
| 施法距离 | 触碰 |
| 专注 | 否 |
| 言语成分 | V |
| 姿势成分 | S |
| 材料成分 | M（价值50gp+的红宝石尘，作为该法术的耗材） |
| 昂贵材料 | 贵材 |
| 持续时间 | 直至被解除 |

## 描述

你所触碰的物件上冒出火焰。这一效应发出20尺半径的明亮光照Bright Light和额外20尺的微光光照Dim Light。法术效应看上去与一般火焰无异，不过其并不会发热，也不需要燃料。你可以掩盖或隐藏火焰，但无法将其闷熄或扑灭。
