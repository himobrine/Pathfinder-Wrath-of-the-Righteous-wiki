# 克敌先击

| 属性 | 值 |
|------|-----|

| 言语成分 |  |
| 姿势成分 | S |
| 材料成分 | M（一把价值1cp+的你熟练的武器） |
| 昂贵材料 | 贵材 |
| 持续时间 | 立即 |

## 描述

你受到一瞬魔法洞见的指引，使用在此法术施法过程中使用的武器进行一次攻击。此次攻击使用你的施法属性而非力量或敏捷进行攻击和伤害掷骰。若攻击造成伤害，则伤害类型可以是光耀或武器一般会造成的伤害类型（由你选择）。戏法升级Cantrip Upgrade。你到达特定等级时，无论你选择光耀伤害还是武器一般会造成的伤害类型，都会额外造成光耀伤害（5级1d6、11级2d6、17级3d6）。
