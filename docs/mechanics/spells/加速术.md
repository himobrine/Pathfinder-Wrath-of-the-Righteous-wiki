| 属性 | 值 |
|------|-----|
| 英文名 | Haste |

| 仪式 | 否 |
| 施法时间 | 动作 |
| 施法距离 | 30尺 |
| 专注 | 是 |
| 言语成分 | V |
| 姿势成分 | S |
| 材料成分 | M（一片甘草根切片） |
| 昂贵材料 |  |
| 持续时间 | 至多1分钟 |

## 描述

指定施法距离内一个你能看见的自愿生物。直至法术终止前，目标速度翻倍；其 AC 获得+2 加值；其进行敏捷豁免时具有优势；其每个回合都可作出一个额外的动作，但该动作只能用于执行攻击（只限一次武器攻击），或者疾行、撤离、躲藏和利用动作。法术终止时，目标将泛起一阵倦意使其失能并且速度为0，并持续至其下一回合结束。
