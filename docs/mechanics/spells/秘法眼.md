| 属性 | 值 |
|------|-----|

| 专注 | 是 |
| 言语成分 | V |
| 姿势成分 | S |
| 材料成分 | M（一点蝙蝠毛皮） |
| 昂贵材料 |  |
| 持续时间 | 至多1小时 |

## 描述

你在施法距离内创造出一个漂浮在空中，不受伤害的隐形魔法眼。你通过精神连接接收秘法眼传来的视觉信息。秘法眼拥有 30 尺黑暗视觉且可以观察任何方向。以一个附赠动作，你可以让秘法眼向任意方向移动 30 尺。固体障碍物会阻挡秘法眼的移动，但它可以穿过任何直径大于 1 寸的缝隙。
