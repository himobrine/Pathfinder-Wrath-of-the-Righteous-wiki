# 欢迎来到开拓者：正义之怒 Wiki

欢迎来到《开拓者：正义之怒》(Pathfinder: Wrath of the Righteous) 的中文攻略 Wiki。本站旨在为玩家提供全面、准确的游戏资料与构筑参考。

- **[职业](classes/index.md)** — 全部职业介绍，包含基础职业、进阶职业与神话道途
- **[游戏机制](mechanics/index.md)** — 核心规则详解（BAB、豁免、法术、专长等）
- **[构筑指南](guides/index.md)** — 热门 Build 分享与搭配思路
- **[BD 制作工具](build-planner.md)** — 在线角色构筑计算器
- **[物品装备](items/index.md)** — 武器、防具、首饰等装备资料
